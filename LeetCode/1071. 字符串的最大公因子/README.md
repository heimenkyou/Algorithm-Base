## 1. 周期性对齐

想象字符串是由一个个相同的“砖块”（最小重复单元）砌成的。

- 如果 `str1` 是由 3 块“砖”组成，`str2` 是由 2 块相同的“砖”组成。
- 那么 `str1 + str2` 就是 5 块砖，`str2 + str1` 也是这 5 块砖。
- **结论：** 只要拼接结果相等，就说明这两个字符串中含有相同的重复单元。

## 2. 为什么是 $gcd$？

一旦确定了它们有共同的因子，剩下的就是纯数学问题：

- `str1` 的长度是 $L_1$，`str2` 的长度是 $L_2$。
- 我们要找的 $X$ 必须能同时整除这两个长度。
- 为了让 $X$ 最长，其长度必然是 $gcd(L_1, L_2)$。

## 3. 一句话

**相等即同源，同源取最大公约长。**

- **第一步：** 检查 `str1 + str2 == str2 + str1`。如果不等，说明它们根本没有共同的重复基底（比如 `LEET` 和 `CODE`），直接返回空。
- **第二步：** 如果相等，说明它们一定由某个基础单元重复而成。此时，最大公约子的长度就是 $gcd(len1, len2)$。
